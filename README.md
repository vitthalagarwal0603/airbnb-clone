## 📌 Project Description
This project is a frontend clone of the Airbnb website created using HTML and CSS.  
It replicates the layout and design of the Airbnb homepage.

---

## ✨ Features
- Modern UI similar to Airbnb
- Property listing layout
- Clean and responsive design
- User-friendly interface

---

## 🛠️ Technologies Used
- HTML5
- CSS3

---

## 📂 Project Structure
- airbnb.html → Main webpage
- CSS / images → Styling and assets

---

## 🎯 Purpose of the Project
This project was developed as part of academic learning to understand:
- Frontend web development
- UI design principles
- Layout structuring
